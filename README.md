# ЖК АКСУ — Онлайн платформа управления

Веб-платформа управления жилым комплексом с авторизацией и общим хранилищем данных.

## 🚀 Деплой на Vercel (5 минут)

### Шаг 1: Загрузите на GitHub

1. Создайте новый репозиторий на [github.com](https://github.com/new)
   - Название: `zhk-aksu-platform`
   - Приватный (Private)
2. Загрузите все файлы проекта в репозиторий

### Шаг 2: Деплой на Vercel

1. Зайдите на [vercel.com](https://vercel.com) и войдите через GitHub
2. Нажмите **"Add New → Project"**
3. Выберите репозиторий `zhk-aksu-platform`
4. В настройках добавьте **Environment Variables**:
   - `APP_PASSWORD` = `ваш_пароль` (пароль для входа в систему)
5. Нажмите **Deploy**

### Шаг 3: Подключите Vercel KV (хранилище данных)

⚠️ **Важно!** Без этого шага данные будут храниться только в памяти и пропадут при перезапуске.

1. В дашборде Vercel откройте ваш проект
2. Перейдите на вкладку **Storage**
3. Нажмите **Create** → **KV (Key-Value Store)**
4. Выберите **Free** план (бесплатно, 256MB)
5. Нажмите **Connect** — Vercel автоматически добавит нужные переменные
6. **Переdeployте** проект (вкладка Deployments → Redeploy)

### Готово! 🎉

Платформа доступна по адресу: `https://zhk-aksu-platform.vercel.app`

## 🔄 Как работает синхронизация

1. При загрузке страницы — данные загружаются с сервера в localStorage
2. При изменении данных — автоматически сохраняются на сервер (debounce 800ms)
3. Каждые 30 секунд — проверка обновлений от других пользователей
4. Индикатор "☁ Сохранено" показывает статус синхронизации
